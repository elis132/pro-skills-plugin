# Voice Critic: Self-Review Rubric

## The Read-Aloud Test

Read the entire piece out loud. Not in your head — out loud (or simulate reading it as speech). Score:

- **Stumbles?** Where you trip = awkward phrasing. Rewrite those sentences.
- **Bored?** Where your energy drops = the writing is flat there. Add a specific detail, a shorter sentence, or cut the section.
- **Robot voice?** Where you start sounding like a press release = AI patterns crept in. Check `references/ai-tells.md`.

## Category Scores

### Voice Consistency
- Does this sound like ONE person wrote it? (Not a committee, not a different person per paragraph)
- Could you describe that person's personality in 2-3 words?
- Are there moments where the "mask slips" and a generic AI voice sneaks in?
- Would this person use ALL of these words? (Watch for vocabulary shifts between sections)

**Fail signals:** The opening is punchy and human, but by paragraph 3 it's drifted into "comprehensive solution" territory. The tone shifts between sections. Jargon appears inconsistently.

### Specificity
- Count the specific details (numbers, names, examples, dates, measurements). You need at least one per paragraph for any persuasive text.
- Is every claim supported by a concrete detail?
- Could any sentence be copy-pasted to a competitor's site? (If yes, it's too generic)

**Fail signals:** Vague claims like "high quality," "fast," "easy to use" without specific proof. Generic benefits that apply to any product in the category.

### Rhythm
- Do sentence lengths vary? (Check: are any three consecutive sentences within 5 words of each other?)
- Is there at least one dramatically short sentence (1-5 words)?
- Is there at least one long, detailed sentence that earns its length?
- Do paragraphs vary in length?

**Fail signals:** Every sentence 15-25 words. Every paragraph 3-4 sentences. Monotone structure throughout.

### Opening Strength
- Does the first sentence make you want to read the second?
- Is the first sentence content (not context)?
- If you deleted the first paragraph, would the piece work better starting at paragraph 2? (If yes, delete paragraph 1)

**Fail signals:** Setup openings, definition openings, "In today's..." openings, summary openings ("In this article...").

### Closing Strength
- Does the ending feel final (not just stopped)?
- Is it memorable?
- Does it avoid "In conclusion..." or summary recaps?
- Does it leave the reader with something — a feeling, a decision, an action?

**Fail signals:** Summaries, hedge-conclusions ("While challenges remain..."), empty calls to action.

### AI-Tell Scan

Go through this checklist item by item:

- [ ] No em-dashes (—). Maximum one per page. Zero is better. This is one of the strongest AI tells.
- [ ] No "whether you're..." inclusive openings
- [ ] No "not just X, but Y" structure
- [ ] No "here's the thing" / "let's dive in" / "it's worth noting" bridges
- [ ] No "leverage/unlock/empower/elevate/seamless/robust/comprehensive"
- [ ] No unnecessary adverbs (truly, incredibly, remarkably, absolutely)
- [ ] No fake enthusiasm ("we're thrilled/excited/passionate")
- [ ] No corporate warmth ("we believe in..." / "we're committed to...")
- [ ] No balanced hedging at the end ("while challenges remain...")
- [ ] No "In today's..." / "In a world where..." / "When it comes to..."
- [ ] No three-item lists with equal-length descriptions
- [ ] Paragraphs vary in length (not all 3-4 sentences)
- [ ] Sentences vary in length (not all 15-25 words)

If more than 2 of these are present, do another editing pass focused only on AI-tell removal.

## The Comparison Tests

### The Swap Test
Remove the brand/product name. Could this text work for a different company in the same category? If yes, it's too generic. Add specifics that only THIS company could claim.

### The Prediction Test  
Cover the second half of each paragraph. Can you guess what it says? If yes, the writing is predictable. Rewrite the second half to surprise.

### The Memory Test
Read the copy once. Close it. What do you remember? If you can recall a specific line or detail, the copy is working. If all you remember is "it was about a monitoring tool" or "it was about a bakery" — it's too forgettable.

### The Skip Test
Be honest: which sections would a real reader skip? Those sections need to either be cut or rewritten to be worth reading. There is no third option.

## Range Check

If you've written multiple pieces in this conversation, compare:
- Do they all open the same way?
- Do they use the same sentence structures?
- Do they have the same rhythm?
- Do they reach for the same vocabulary?

Different audiences and contexts should produce writing that feels different — not just in content but in structure, rhythm, and personality. A SaaS landing page should read differently from a bakery's about page, which should read differently from a legal brief.
